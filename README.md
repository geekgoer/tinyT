# tinyT  v0.0.1

# usage
## 1. no filename 
`./tinyT`
## 2. use filename
`./tinyT <filename>`

# Save 
## 1. no filename
1. `CTRL + S`
2. type in filename

## 2.has filename
1. CTRL + S

# Quit
push `CTRL + Q` 2 times

# End the current session behavior
push `Esc`
